webstorm 常用快捷键
ctrl + G 查找第几行
ctrl + F 查找相同单词
ctrl + F 在浏览器开发者工具中搜索对象的某一个属性
ctrl + alt + L 解压缩CSS文件/格式化代码
alt + F2 在谷歌浏览器中运行
选中多行-->ctrl + shift + 右键-->删除选中-->加注释-->格式化代码
选中多行--> shift + end-->ctrl + x-->加''-->ctrl + v-->home键 做大量字符串拼接
shift + enter 浏览器控制台中测试代码用作 换行
ctrl + 鼠标左键 jQuery中查找对应单词的方法
在ff浏览器中 首先安装Title 3D，安装完成，使用ctrl + shift + L，可查看3d视图
